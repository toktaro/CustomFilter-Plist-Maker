# CustomFilter Plist Maker

Glyphs3のカスタムフィルタ（CustomFilter）用plistファイルを作成・編集するブラウザベースのツールです。

![Screenshot](screenshot.png)

## Live Demo

[https://toktaro.github.io/CustomFilter-Plist-Maker/](https://toktaro.github.io/CustomFilter-Plist-Maker/)



## 機能

- **複数フィルタ対応**: 1つのplistに複数のフィルタを含めて管理
- **フィルタ操作**: 追加・編集・削除・ドラッグ＆ドロップで並び替え
- **文字入力**: テキストを直接入力・ペースト、またはテキストファイル（.txt）をドラッグ＆ドロップ
- **既存plistの読み込み**: 既存のplistファイルを読み込んで編集（複数フィルタにも対応）
- **グリフ名対応**: Unicode形式（uni3042）とPostScriptグリフ名（space, Eth, A等）の両方に対応
- **Unicode順ソート**: 登録した文字をUnicodeコードポイント順に並び替え
- **重複の自動除去**: 同じ文字を複数回入力しても自動的に1つにまとめる
- **個別文字の削除**: 各文字にマウスを合わせると×ボタンで個別に削除可能
- **書き出し**: 個別フィルタまたは全フィルタを一括でplistとしてダウンロード

## 使い方

### 基本的な流れ

1. `index.html` をブラウザで開く
2. 「新しいフィルタを追加」でフィルタを作成
3. フィルタ名を入力
4. 「文字を追加」から文字を入力（テキスト入力またはtxtファイルをドロップ）
5. 必要に応じて複数のフィルタを追加
6. 「すべてを1つのplistとしてダウンロード」でファイルを保存

### 既存plistの編集

1. 「既存plistを読み込む」ボタンからplistファイルを選択
2. 複数フィルタがある場合は、すべてのフィルタが一覧に追加される
3. 各フィルタを選択して編集
4. 書き出し

### 個別書き出し

各フィルタの「書き出し」ボタンをクリックすると、そのフィルタのみを個別のplistとしてダウンロードできます。  
ファイル名は「元のファイル名-フィルタ名.plist」形式になります。

### Glyphs3での使用

ダウンロードしたplistファイルをglyphsファイルと同じディレクトリ（フォルダ）、
または以下のディレクトリに配置：

```
(ユーザー)~/Library/Application Support/Glyphs 3/Filters/
```

Glyphs3を再起動すると、サイドバーのフィルタ下に追加されます。

## 技術仕様

- **動作環境**: モダンブラウザ（Chrome, Safari, Firefox, Edge）
- **依存関係**: なし（フレームワーク不使用）
- **ファイル形式**: 単一HTMLファイル（CSS・JavaScript埋め込み）

### グリフ名の形式

文字入力時はUnicode形式に変換されます。plist読み込み時はグリフ名をそのまま保持します：

| 入力 | 保存されるグリフ名 | 備考 |
|------|------------------|------|
| 茨 | uni8328 | 文字入力からはUnicode形式へ変換 |
| あ | uni3042 | |
| A | uni0041 | |
| （plistから読込）| space | PostScriptグリフ名をそのまま保持 |
| （plistから読込）| Eth | |
